# Tab Components
> 以component切換及針對input三種輸入(select、radio、text)功能為練習目標。

### 簡介
Tab 功能 :
* 應用 <component v-bing:is> 及 reactive 雙向綁定
* <keep-alive> 記憶Tab切換前的狀態
* 使用 provide, inject 功能

Select 部分 :
* v-model 及 reactive 資料雙向綁定

Radio 部分 :
* 非同步取得資料並處理後渲染到畫面上

Text 部分 :
* 實作簡易搜尋功能
